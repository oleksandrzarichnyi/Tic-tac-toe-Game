Stack: React, Zustand, Tailwind

Two-player game with chat functionality.

Features: 
1. Turn system: players cannot make a move unless it is their turn. Game state is managed using Zustand.
2. Game reset: either player can reset the game using the "Reset" button.
3. Scoreboard: win tracking and score calculation are handled via Zustand.
4. Auto restart: when a player wins, the game automatically restarts after 5 seconds.
5. Chat: both players can exchange messages in real time using the built-in chat.

Run the project locally: 
npm run dev
